# 구구단 테스트

브라우저에서 바로 실행되는 구구단 퀴즈 웹 앱입니다.

## 기능

- **단 선택**: 2단~9단 중 복수 선택 가능 (최소 1개)
- **문제 수 모드**: 10문제 / 20문제 / 무제한
- **실시간 통계**: 정답 수, 오답 수, 정답률 표시
- **진행률 바**: 유한 모드에서 진행 상황 시각화
- **결과 화면**: 완료 후 점수 및 피드백 이모지 표시
- **키보드 지원**: Enter 키로 정답 제출 가능

## 점수 기준

| 정답률 | 평가 |
|--------|------|
| 90% 이상 | 완벽해요! |
| 70% 이상 | 잘했어요! |
| 50% 이상 | 괜찮아요! |
| 50% 미만 | 더 연습해요! |

## 실행 방법

별도 설치 없이 `index.html`을 브라우저에서 열면 바로 실행됩니다.

## 파일 구조

```
.
├── index.html   # 앱 구조 및 마크업
├── style.css    # 스타일
└── app.js       # 퀴즈 로직
```
